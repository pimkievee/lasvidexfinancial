# SEO Setup Checklist

Code placeholders are ready for indexing and analytics. Replace the placeholder values once the accounts are created.

## Google Search Console

1. Add `https://lasvidexfinancial.com/` as a property.
2. Choose HTML meta tag verification.
3. Replace `GOOGLE_SEARCH_CONSOLE_VERIFICATION_PLACEHOLDER` in the site HTML with the provided value.
4. Submit `https://lasvidexfinancial.com/sitemap.xml`.

## Bing Webmaster Tools

1. Add `https://lasvidexfinancial.com/`.
2. Choose meta tag verification.
3. Replace `BING_WEBMASTER_VERIFICATION_PLACEHOLDER` in the site HTML with the provided value.
4. Submit `https://lasvidexfinancial.com/sitemap.xml`.

## Google Analytics 4

1. Create a GA4 web data stream for `lasvidexfinancial.com`.
2. Replace `G-PLACEHOLDER` with the real GA4 measurement ID.
3. Add the GA4 script snippet before `</head>` on each public HTML page.

## Google Business Profile

1. Confirm business name, service areas, phone number, website URL, and business category.
2. Link the profile to `https://lasvidexfinancial.com/`.
3. Add services that match the site: financial planning, insurance planning, retirement planning, estate planning, RESP, TFSA, RRSP, FHSA, and business owner planning.
4. Add posts that link to the Resources hub and the Whitby landing page.
